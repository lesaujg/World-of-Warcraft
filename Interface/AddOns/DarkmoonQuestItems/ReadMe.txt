Description: This addon will show you whether you completed the quests associated with the Darkmoon Faire quest items.
Just type in the chat: /dqi, and this addon will show you informations about the Darkmoon Faire quest items.

Installation: Put the DarkmoonQuestItems folder into the: World of Warcraft\_retail_\Interface\AddOns folder.

Commands:
- /dqi - Shows you informations about the Darkmoon Faire quest items.
